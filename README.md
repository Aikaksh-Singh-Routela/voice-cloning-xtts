# 🎤 Voice Cloning XTTS

[![Python](https://img.shields.io/badge/Python-3.11+-blue.svg)](https://python.org)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Coqui](https://img.shields.io/badge/TTS-Coqui%20XTTS--v2-FF6B6B)](https://coqui.ai)

Voice cloning app using **Coqui XTTS-v2** — clone voices from short audio samples and generate speech in 10+ languages.

## 🚀 Features

- 🎤 **Voice Cloning** — Clone voices from 3-15 second WAV samples
- 🌍 **Multi-language** — Supports English, Spanish, French, German, Hindi, Japanese, and more
- 📥 **Download Audio** — Save generated speech as WAV
- ⚡ **Fast Generation** — ~2-5 seconds per generation

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **TTS Engine:** Coqui XTTS-v2
- **Audio Processing:** soundfile, PyTorch
- **Deployment:** Streamlit Cloud

## 📦 Installation

```bash
git clone https://github.com/Aikaksh-Singh-Routela/voice-cloning-xtts.git
cd voice-cloning-xtts
pip install -r requirements.txt
streamlit run app.py
🎮 How to Use
Upload a WAV file (3-15 seconds of a single voice)

Enter text in any supported language

Click "Generate Speech"

Listen and download the cloned voice
